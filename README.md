# 🛍️ CV. Kawanua Plastik — Landing Page

A responsive, mobile-first landing page for CV. Kawanua Plastik, a local plastic supplier based in Sukabumi, West Java — the company's first-ever digital presence.

## ✨ Features

- Clean promotional landing page with no framework dependencies
- Smooth scroll-triggered fade-in animations (IntersectionObserver)
- SVG product illustrations (hand-crafted, no external assets)
- WhatsApp inquiry form with auto-formatted message
- Fully responsive — mobile-first layout
- Fixed navigation with blur backdrop
- Interactive product grid (6 categories)

## 🛠️ Tech Stack

| Component     | Technology                     |
|---------------|--------------------------------|
| Markup        | HTML5                          |
| Styling       | Vanilla CSS (CSS Variables)    |
| Scripting     | Vanilla JavaScript             |
| Typography    | Google Fonts (Playfair Display + Inter) |
| Illustrations | Inline SVG                     |
| Hosting       | GitHub Pages / Static Host     |

## 📁 Project Structure

```
kawanua-plastik/

```

## 🚀 Getting Started

### Run Locally

No build tools or dependencies required.

```bash
git clone https://github.com/your-username/kawanua-plastik.git
cd kawanua-plastik
open index.html
```

Or simply open `index.html` directly in any browser.

### Deploy to GitHub Pages

1. Push the repository to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Site will be live at `https://your-username.github.io/kawanua-plastik`

## ⚙️ Configuration

Before deploying, update the following in `index.html`:

| Item | Location | Description |
|------|----------|-------------|
| WhatsApp number | `wa.me/62XXXXXXXXXX` (×2) | Replace with the company's active WA number |
| Address | Contact & About sections | Update if the address changes |
| Rating / stats | Stats strip & cards | Update with latest data |
